# 视觉规范

本文是项目的默认页面风格基线，适用于所有后台页面。

## 1. 风格定位

- 浅色后台、克制简洁、信息清晰
- 蓝色主强调、白色卡片容器
- 参考页面：离线视图分析、离线新建任务

## 2. 设计 Token

### 2.1 颜色

```css
--color-primary: #2d83f5;
--color-primary-hover: #225fdf;
--color-text: #273446;
--color-text-secondary: #516377;
--color-text-muted: #6f8095;
--color-border: #dbe4ef;
--color-bg-page: #f3f6fb;
--color-bg-panel: #ffffff;
--color-danger: #ef7270;
--color-danger-strong: #ff4d4f;
--color-success: #56bf94;
--color-warning: #f2a85f;
```

使用原则：
- 蓝色：主操作、当前状态、重点信息
- 红色：删除、危险、失败
- 橙色：告警、待处理
- 绿色：成功、正常、启用

### 2.2 字体

```css
font-family: "Source Sans Pro", "PingFang SC", "Microsoft YaHei", sans-serif;
```

| 场景 | 字号 | 字重 |
|------|------|------|
| 页面标题 / 面包屑当前页 | 18px | 700 |
| 面板标题 | 16px | 700 |
| 筛选项标签 / 按钮文字 | 14px | 600 |
| 表格正文 | 14px | 400 |
| 表头 | 12px | 700 |
| 辅助文字 / 行内操作 | 12px | 400 |

### 2.3 尺寸

| 元素 | 高度 | 圆角 |
|------|------|------|
| 筛选输入框 / 按钮 | 40px | 6px |
| 分页按钮 | 32px | 8px |
| 状态标签 | 24-28px | 4px |
| 面板 | - | 10-14px |

## 3. 页面骨架

- 左侧导航（展开 252px / 收起 68px）
- 右侧内容区：顶部白底顶栏 + 白色面板
- 全局浅灰蓝底色，模块间用间距分层

## 4. 列表页规范

### 4.1 结构

```
顶栏（面包屑）
└── 白色面板
    ├── 工具栏（左：主操作 / 右：筛选）
    ├── 表格容器
    └── 分页
```

### 4.2 按钮

| 类型 | 用途 | 样式 |
|------|------|------|
| Primary | 主要操作（查询、新增） | 蓝底白字 |
| Secondary | 次级操作（重置、导入、导出） | 白底描边 |
| Danger | 高风险操作（批量删除） | 红底白字 |
| Text Link | 行内操作 | 透明底、蓝/红文字 |

规则：
- 高度 40px、最小宽度 96-104px、字号 14px
- 行内操作：普通用蓝色链接，删除用红色链接
- 不使用胶囊小按钮

### 4.3 表格

- 白底、1px 浅边框、10px 圆角
- 表头：12px / 700、浅底色
- 单元格：padding 12px 14px
- 超宽时表格容器内滚动，操作列固定右侧

### 4.4 状态标签

| 状态 | 文字色 | 背景色 |
|------|--------|--------|
| 成功 / 在线 | success | success-soft |
| 失败 / 离线 | danger | danger-soft |
| 告警 / 待处理 | warning | warning-soft |
| 默认 | 中性灰 | 浅灰 |

### 4.5 筛选区

顺序：状态 → 类型 → 关键字 → 查询 → 重置 → 刷新
- 输入框高度 40px，下拉宽度 140px，搜索框宽度 220px
- 不额外套卡片

## 5. 创建页 / 详情页规范

- 白色面板内按 section 分段
- 标签列约 76px，行间距 16-18px
- 上传区使用虚线框
- 激活态只用蓝色

## 6. 仪表盘规范

- 浅底色、白卡片、蓝色主强调
- 允许数据卡片、图表
- 禁止深色科技风、大面积渐变

## 7. 禁止事项

- 新页面单独定义主色
- 混用多套主蓝
- 大面积重阴影、深色科技风
- 行内操作做成胶囊按钮
- 筛选区再包显眼卡片
- 同类页面出现不同圆角/输入框/按钮高度
